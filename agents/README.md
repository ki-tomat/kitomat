## Hermes

Hermes is for release, handover and communication drafts after review facts exist.

- [Hermes system prompt](hermes-release/system-prompt.md)
- [Hermes step by step](hermes-release/hermes-step-by-step.md)

## Boundary

Agents do not approve, merge, release, publish or replace human expert review.
