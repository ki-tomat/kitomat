# Gegenüber kennen
*12 Stakeholder-Profile für KI-Projekte in KMU, empirisch belegt*

## Wofür ist das?
Das Paket enthält 12 empirisch begründete Stakeholder-Archetypen-Profile, die typische Entscheidungslogiken, Kommunikationspräferenzen und KI-Adoptionshaltungen von Akteur:innen bei der Einführung von KI in KMU beschreiben. Es dient als Referenz für die Erstellung von Prompts, Szenarien und Bedarfsanalysen in KI-Beratungsprojekten.

## Für wen?
- KI-Berater:innen und Consultants, die Stakeholder-Kommunikation vorbereiten
- Prompt-Engineers, die kontextspezifische Prompts für spezifische Zielgruppen entwickeln
- Projektleitungen, die Change-Management-Maßnahmen auf Zielgruppen abstimmen

## Archetypen in diesem Paket

| Nr. | Archetyp | Kerndimension |
|---|---|---|
| 01 | CEO / Geschäftsführer | Strategische Relevanz, Wettbewerbsvorteil, Eigentümerlogik |

| 04 | Legal / Datenschutz | Risikoklassifizierung, DSGVO, EU AI Act |
| 05 | Produktmanagement | Problem-Solution-Fit, Stage-Gate, Responsible Use |
| 06 | IT-Leitung / CTO | Task-Technology-Fit, Governance, Business-IT-Alignment |
| 07 | Vertrieb / Sales | Unmittelbarer Nutzen, Workflow-Fit, Beziehungskompetenz |
| 08 | Controlling | Steuerungsqualität, Nachvollziehbarkeit, Audit-Sicherheit |
| 09 | Investor (VC / PE) | Skalierbarkeit, EBITDA-Impact, Exit-Fähigkeit |
| 10 | Advisory Board / Beirat | Strategische Governance, „Noses In, Fingers Out" |
| 11 | Betriebsrat | Mitbestimmung, BetrVG, algorithmische Kontrolle |
| 12 | Service / Sachbearbeitung | TAM, STARA, algorithmische Kontrolle am Arbeitsplatz |

## Wichtiger Hinweis
Die Archetypen sind Stakeholder-Linsen, keine Persönlichkeitsprofile. Alle Profile sind empirisch begründete Typenbeschreibungen auf Basis peer-reviewter Literatur — keine Aussagen über Einzelpersonen. Bei kritischen Entscheidungen immer die Originalquellen prüfen.

## Verknüpfung
Dieses Paket bildet die empirische Quellenbasis für das Schwesterartefakt **Perspektive wechseln** (jk-ki-stakeholder-persona-framework), das tiefere Persona-Simulationen ermöglicht.
