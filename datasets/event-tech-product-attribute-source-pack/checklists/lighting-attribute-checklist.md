|---|---|---|---|---|
| Marke / Hersteller | Hersteller oder Marke | Pflicht | offiziell / werbetext | |
| Modell | Modellbezeichnung | Pflicht | offiziell | |
| Produktname | verkaufsfaehiger Produktname | Pflicht | offiziell / werbetext | |
| SKU / Artikelnummer | interne oder Shop-Artikelnummer | Pflicht | offiziell / intern | |
| Hersteller-MPN | Manufacturer Part Number | Pflicht | offiziell | |
| GTIN | globale Artikelnummer | Pflicht | offiziell | |
| EAN | europaeische Artikelnummer | Pflicht | offiziell | |
