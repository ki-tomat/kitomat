# Claim Check Template

Dieses Template hilft, Marketingclaims, technische Aussagen und abgeleitete Nutzenargumente zu bewerten.

## Claim-Tabelle

| Claim / Aussage | Quelle | Quellenvertrauensstufe | Bewertung | Empfehlung | Uebernahme |
|---|---|---|---|---|---|
| ... | ... | offiziell / werbetext / extern / unklar | belegt / abgeleitet / Annahme / pruefen / nicht verwenden | ... | uebernehmen / pruefen / nicht uebernehmen |

## Bewertungslogik

| Bewertung | Bedeutung |
|---|---|
| `belegt` | Aussage ist durch offizielle Quelle gestuetzt. |
| `abgeleitet` | Aussage ist plausibel aus Daten ableitbar, aber kein direkter Herstellerfakt. |
| `Annahme` | Aussage ist wahrscheinlich, aber nicht ausreichend belegt. |
| `pruefen` | Aussage muss fachlich geprueft werden. |
| `nicht verwenden` | Aussage darf nicht veroeffentlicht werden. |

## Claims, die nie ungeprueft veroeffentlicht werden duerfen

Diese Claims muessen immer belegt oder fachlich geprueft werden. Ohne belastbare Grundlage gehoeren sie in `Nicht veroeffentlichen ohne Pruefung`.

| Claim / Formulierung | Risiko | Standard-Umgang |
|---|---|---|
| universell kompatibel | zu breite Kompatibilitaetsbehauptung | nur mit konkreten Systemen/Protokollen nennen |
| wetterfest | falsche Outdoor-Erwartung | nur mit offizieller Schutzart und Herstellerangabe |
| fuer alle Anwendungen geeignet | Ueberversprechen | konkrete Einsatzbereiche nennen |
| garantiert sicher | Sicherheitsfreigabe wird suggeriert | nicht verwenden |
| sicher fuer alle Buehnen | gefaehrliche Pauschalaussage | nicht verwenden |
| professionelle Buehnentauglichkeit | unklarer Leistungs-/Einsatzclaim | nur konkret und belegt formulieren |
| garantiert latenzfrei | technische Garantie ohne Beleg | nicht verwenden |
| wartungsfrei | kann Pflege-/Wartungspflichten verdecken | nur belegt verwenden |
| ohne Einschraenkungen nutzbar | Grenzen werden verschwiegen | nicht verwenden |
| ersetzt fachliche Planung | gefaehrliche Freigabe | nicht verwenden |
| reicht fuer feste Personenzahl | situationsabhaengige PA-/Raumfrage | nur als Pruefpunkt, nicht als Garantie |
| fuer Outdoor geeignet | kann Schutzart/Umgebung falsch darstellen | nur bei offizieller Freigabe |
| Montage ueber Personen freigegeben | sicherheitskritische Aussage | nur mit offizieller Dokumentation und Fachpruefung |
| alle Steuerungen kompatibel | technische Kompatibilitaet wird ueberdehnt | nur konkrete Protokolle/Geraete nennen |
