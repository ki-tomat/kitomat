# Product Identity Required Fields

Diese Felder gelten produktuebergreifend als Pflichtangaben fuer Marker-, Modell- und Produktidentifikation.

| Attribut | Beschreibung | Prioritaet | Quelle / Vertrauensstufe | Status / Pruefhinweis |
|---|---|---|---|---|
| Marke / Hersteller | Hersteller oder Marke des Produkts | Pflicht | offiziell / werbetext | |
| Modell | Modellbezeichnung | Pflicht | offiziell | |
| Produktname | verkaufsfaehiger Produktname | Pflicht | offiziell / werbetext | |
| SKU / Artikelnummer | interne oder shopbezogene Artikelnummer | Pflicht | offiziell / intern | |
| Hersteller-MPN | Manufacturer Part Number | Pflicht | offiziell | |
| GTIN | globale Artikelnummer, falls vorhanden | Pflicht | offiziell | |
| EAN | europaeische Artikelnummer, falls vorhanden | Pflicht | offiziell | |

## Hinweis

Wenn GTIN oder EAN fuer ein Produkt nicht existiert oder nicht bereitgestellt wird, darf kein Wert erfunden werden. Das Feld bleibt dann ein offener Pruefpunkt.
