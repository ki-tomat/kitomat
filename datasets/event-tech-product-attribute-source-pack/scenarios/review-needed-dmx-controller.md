# Nachbearbeitbares Szenario: DMX-Controller mit unvollstaendigen Quellen

## Fall

Ein Shop moechte einen DMX-Controller beschreiben. Es liegen nur Werbetext und ein kurzer Produktseitentext vor, aber kein Manual und kein DMX-Dokument.

## Quellen

| Quelle | Vertrauensstufe |
|---|---|
| Herstellerwebsite | werbetext |
| Produktteaser | werbetext |
| fehlendes Manual | unklar |

## Erwartetes Ergebnis

- keine fertige technische Freigabe
- offene Pruefpunkte
- Dokumente nachfordern
- Claims vorsichtig formulieren
- CMS-Import technischer Daten nur nach Pruefung
