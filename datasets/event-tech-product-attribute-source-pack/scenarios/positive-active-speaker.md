# Positives Szenario: Aktiver Lautsprecher

## Fall

Ein KMU-Shop moechte einen aktiven 12-Zoll-Lautsprecher fuer kleine bis mittlere Veranstaltungen beschreiben.

## Quellen

| Quelle | Vertrauensstufe |
|---|---|
| offizielles Datenblatt | offiziell |
| Bedienungsanleitung | offiziell |
| Herstellerbeschreibung | werbetext |
| synthetische Supportfragen | extern / synthetisch |

## Erwartetes Ergebnis

- strukturierte Produktdaten
- CMS-/PIM-Feldvorschlag
- klare Produktbeschreibung
- FAQ
- Supportantworten
- Claim-Check
- keine festen Personenzahl-Garantien
- keine Outdoor-Eignung ohne Beleg
