# Failure Modes

- Der Prompt wird generisch, wenn Zielgruppe oder Kontext fehlen.
- Der Prompt kann Quellen erfinden, wenn Quellen nicht begrenzt werden.
- Der Prompt kann echte Daten verarbeiten, wenn Nutzer sie einfuegen; das ist im MVP nicht erlaubt.
- Der Prompt darf nicht fuer rechtliche, medizinische, HR-, Finanz- oder Bildungsentscheidungen ueber Menschen genutzt werden.
- Der Prompt ersetzt keine fachliche Endpruefung.
