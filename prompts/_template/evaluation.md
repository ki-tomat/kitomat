# Evaluation

## Qualitaetskriterien

- [ ] Zielgruppe ist klar.
- [ ] Problem ist konkret.
- [ ] Eingaben reichen fuer ein brauchbares Ergebnis.
- [ ] Outputformat wird eingehalten.
- [ ] Keine erfundenen Quellen oder Zahlen.
- [ ] Keine unzulaessigen Daten.
- [ ] Menschliche Pruefung bleibt sichtbar.

## Prueffragen

1. Woran erkennt ein Experte, dass das Ergebnis gut ist?
2. Welche Annahmen muessen geprueft werden?
3. Welche Quellen oder Fachregeln stuetzen das Ergebnis?
