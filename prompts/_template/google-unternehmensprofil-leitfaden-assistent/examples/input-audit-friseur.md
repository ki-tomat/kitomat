# Beispielinput: Bestehendes Google Unternehmensprofil prüfen

## Zweck dieser Beispieldatei

Diese Datei ist ein realistischer Beispielinput für den Prompt `google-unternehmensprofil-leitfaden-assistent`.

Der Input simuliert den Fall, dass ein kleines lokales Unternehmen bereits ein Google Unternehmensprofil besitzt, dieses aber bisher nur unregelmäßig gepflegt wurde.

Der Leitfaden-Prompt soll auf Basis dieser Angaben den passenden Modus wählen:

```text
Modus A: Bestehendes Profil überprüfen
```

Alle Angaben sind fiktiv und dienen ausschließlich als Testmaterial für die Prompt-Bibliothek.
