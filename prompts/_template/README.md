## Nutzung

1. Eingaben vorbereiten.
2. `prompt.md` verwenden.
3. Ergebnis anhand `evaluation.md` pruefen.
4. Grenzen aus `failure-modes.md` beachten.

## Szenario-Triade

### Positiv

Synthetisches Szenario, in dem der Prompt funktioniert.

### Nachbearbeitbar

Szenario, das weitere Expertenangaben, Quellen oder bessere Eingaben braucht.

### Negativ

Szenario, das nicht genutzt werden darf oder zu falschen Ergebnissen fuehrt.

## Trust-Hinweis

Dieses Paket ersetzt keine fachliche Pruefung. Vor produktiver Nutzung muss das Ergebnis menschlich kontrolliert werden.
