# FinOps Assessment Worksheet

## Nutzung

Dieses Worksheet dient zur Standortbestimmung. Bewerte jeden Bereich auf einer Skala von 0 bis 5.

Skala:

- 0 = nicht vorhanden
- 1 = ad hoc
- 2 = wiederholbar
- 3 = definiert
- 4 = gemessen und gesteuert
- 5 = skaliert und optimiert
