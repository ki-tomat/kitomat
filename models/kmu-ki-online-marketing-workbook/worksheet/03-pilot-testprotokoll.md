## F. Bewertung des Ergebnisses

### Kategorie 1: Positiv validierbar

Ankreuzen, wenn:

- [ ] Ergebnis ist brauchbar
- [ ] Ziel wurde erreicht
- [ ] Aufwand ist realistisch
- [ ] keine kritischen Risiken erkennbar
- [ ] menschliche Freigabe ist möglich
- [ ] Routine kann wiederholt werden

**Bewertung:** [ ] positiv validierbar

### Kategorie 2: Nachbearbeitbar

Ankreuzen, wenn:

- [ ] Grundidee ist brauchbar
- [ ] Ergebnis braucht fachliche Korrektur
- [ ] Tonalität muss angepasst werden
- [ ] Informationen fehlen
- [ ] Prompt / Ablauf muss verbessert werden
- [ ] menschliche Prüfung ist zwingend nötig

**Bewertung:** [ ] nachbearbeitbar

### Kategorie 3: Negativ / nicht nutzbar

Ankreuzen, wenn:

- [ ] Ergebnis ist zu generisch
- [ ] Ergebnis enthält falsche Aussagen
- [ ] Ergebnis enthält erfundene Quellen
- [ ] Ergebnis nutzt ungeeignete Daten
- [ ] Aufwand ist zu hoch
- [ ] Risiko ist zu hoch
- [ ] Ziel wurde verfehlt

**Bewertung:** [ ] negativ / nicht nutzbar
